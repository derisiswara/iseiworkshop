Bagian 1: Regresi Linear Sederhana

Apa itu Regresi?

  • Regresi adalah metode statistik untuk memodelkan hubungan antara variabel
  • Variabel dependen (Y): variabel yang ingin kita prediksi/jelaskan
  • Variabel independen (X): variabel yang digunakan sebagai prediktor
  • Model: \(Y = \beta_0 + \beta_1 X + \varepsilon\)

Komponen Model Regresi

Komponen Simbol Makna
Intercept \(\beta_0\) Nilai Y ketika X = 0
Slope \(\beta_1\) Perubahan Y untuk setiap kenaikan 1 unit X
Error \(\varepsilon\) Selisih antara nilai prediksi dan nilai aktual

Tujuan: Menemukan garis “terbaik” yang meminimalkan jumlah kuadrat error (Ordinary Least Squares / OLS)

Korelasi vs Regresi

  • Korelasi: mengukur kekuatan dan arah hubungan linear antara dua variabel
  • Regresi: memprediksi nilai satu variabel berdasarkan variabel lain
  • Korelasi bukan kausalitas!
  • Korelasi simetris (\(r_{xy} = r_{yx}\)), regresi tidak simetris

Membuat Model Regresi Sederhana

m1 <- lm(kepuasan_hidup ~ kepuasan_finansial, data = dat)

Formula: Y ~ X dibaca “Y diregresikan terhadap X”

  • ~ artinya “diprediksi oleh”
  • Sisi kiri: variabel dependen
  • Sisi kanan: variabel independen

Membaca Output summary()

  1. Call: formula model yang digunakan
  2. Residuals: ringkasan distribusi residual
  3. Coefficients:
    • Estimate: nilai koefisien (\(\beta\))
    • Std. Error: standard error koefisien
    • t value: statistik uji (\(\beta / SE\))
    • Pr(>|t|): p-value
  4. R-squared: proporsi varians yang dijelaskan model
  5. F-statistic: uji signifikansi keseluruhan model

Bagian 2: Regresi Linear Berganda

Mengapa Regresi Berganda?

  • Model sederhana hanya menggunakan satu prediktor
  • Realita: banyak faktor mempengaruhi suatu outcome
  • Regresi berganda memungkinkan kita:
    • Mengontrol variabel lain (confounders)
    • Melihat kontribusi unik setiap prediktor
    • Meningkatkan prediksi

Formula Regresi Berganda

\[Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \ldots + \beta_p X_p + \varepsilon\]

Di R:

lm(Y ~ X1 + X2 + X3, data = ...)

+ menambahkan prediktor ke model (bukan operasi matematika)

Asumsi Regresi Linear

  1. Linearitas: hubungan antara X dan Y linear
  2. Independensi: residual saling independen
  3. Homoskedastisitas: varians residual konstan
  4. Normalitas: residual berdistribusi normal
  5. Multikolinearitas: prediktor tidak terlalu berkorelasi satu sama lain

Bagian 3: Variabel Kategorikal dan Pelaporan

Variabel Kategorikal sebagai Prediktor

  • R secara otomatis membuat dummy variables dari faktor
  • Kategori referensi: level pertama (alfabet) dari faktor
  • Koefisien dummy: perbedaan rata-rata dibanding referensi

  mutate(
    jenis_kelamin = factor(jenis_kelamin),
    negara = factor(negara)
  )

Pelaporan Hasil Regresi (APA Style)

Cara melaporkan hasil regresi dalam format APA:

Regresi linear berganda dilakukan untuk menguji pengaruh kepuasan finansial, kebebasan memilih, religiusitas, dan usia terhadap kepuasan hidup. Model secara keseluruhan signifikan, \(F(4, N) = ..., p < .001\), dan menjelaskan …% varians kepuasan hidup (\(R^2_{adj} = ...\)). Kepuasan finansial (\(\beta = ..., p < .001\)) dan kebebasan memilih (\(\beta = ..., p < .001\)) merupakan prediktor signifikan.

Tips Pelaporan

  • Laporkan \(R^2\) adjusted (bukan \(R^2\) biasa)
  • Gunakan unstandardized coefficients (\(B\)) ATAU standardized (\(\beta\)) — pilih satu dan konsisten
  • Laporkan F-test keseluruhan model
  • Sebutkan jumlah observasi
  • Jika ada variabel yang tidak signifikan, tetap laporkan

Ringkasan Sesi 2

Apa yang Sudah Kita Pelajari

  • Regresi sederhana: satu prediktor, lm(Y ~ X)
  • Regresi berganda: banyak prediktor, lm(Y ~ X1 + X2 + ...)
  • Interpretasi: koefisien, \(R^2\), p-value
  • Diagnostik: residual plots, VIF, asumsi
  • Variabel kategorikal: dummy variable, factor()
  • Interaksi: X1 * X2
  • Perbandingan model: ANOVA, AIC
