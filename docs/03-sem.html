Apa itu SEM?

  • Structural Equation Modeling = teknik statistik multivariat yang menggabungkan analisis faktor dan regresi
  • Memungkinkan pengujian hubungan simultan antara beberapa variabel
  • Dapat menguji variabel laten (tidak langsung teramati) melalui indikator-indikator terukur
  • Terdiri dari dua komponen: measurement model dan structural model

Variabel Laten & Manifes

Istilah Penjelasan
Variabel Laten Konstruk yang tidak dapat diukur langsung (simbol oval)
Variabel Manifes/Indikator Item kuesioner atau pengukuran langsung (simbol kotak)
Variabel Eksogen Variabel independen — tidak diprediksi oleh variabel lain
Variabel Endogen Variabel dependen — diprediksi oleh variabel lain

Measurement Model vs Structural Model

  • Measurement Model: hubungan antara variabel laten dan indikator-indikatornya
    • “Seberapa baik indikator mengukur konstruk?”
  • Structural Model: hubungan antar variabel laten
    • “Bagaimana konstruk saling mempengaruhi?”

Note

SEM menguji keduanya secara bersamaan, sehingga memberikan gambaran yang lebih komprehensif dibanding regresi biasa.

Two-Step Approach

Anderson & Gerbing (1988) merekomendasikan:

  1. Langkah 1 — CFA: Evaluasi measurement model terlebih dahulu
    • Apakah indikator mengukur konstruk dengan baik?
    • Validitas konvergen & diskriminan
  2. Langkah 2 — Full SEM: Tambahkan jalur struktural
    • Uji hipotesis antar konstruk

Important

Jangan langsung menjalankan full SEM tanpa memastikan measurement model sudah baik! Jika pengukuran buruk, kesimpulan struktural tidak dapat dipercaya.

No Hipotesis Path

H2 PMB berpengaruh positif terhadap HV HV ~ PMB

Sintaks lavaan

Operator Makna Contoh
=~ “diukur oleh” (measurement) PMB =~ PMB1 + PMB2
~ “diregresikan terhadap” (structural) ELOY ~ PMB + HV
~~ kovarians/varians HV ~~ ELOY
:= defined parameter (mediasi) indirect := a*b

Langkah 1: Spesifikasi Model CFA

cfa_model <- '
  PMB  =~ PMB1 + PMB2 + PMB3 + PMB4
  HV   =~ HV1 + HV2 + HV3 + HV4
  ELOY =~ ELOY1 + ELOY2 + ELOY3
'

Estimasi CFA

cfa_fit <- cfa(cfa_model, data = dat)
  • Estimator default: ML (Maximum Likelihood)
  • Secara otomatis menetapkan loading pertama = 1 untuk identifikasi model
  • cfa() hanya mengestimasi measurement model (tanpa jalur struktural)

Composite Reliability (CR) & AVE

std_loadings <- parameterEstimates(cfa_fit, standardized = TRUE) |>
  filter(op == "=~") |>
  select(lhs, std.all)

reliabilitas <- std_loadings |>
  group_by(Konstruk = lhs) |>
  summarize(
    n_items = n(),
    sum_lambda = sum(std.all),
    sum_lambda2 = sum(std.all^2),
    AVE = sum(std.all^2) / n(),
    CR = sum_lambda^2 / (sum_lambda^2 + (n() - sum(std.all^2))),
    .groups = "drop"
  )

Langkah 2: Full Structural Model

Setelah CFA memadai, tambahkan jalur struktural dengan label untuk analisis mediasi:

sem_model <- '
  # Measurement model
  PMB  =~ PMB1 + PMB2 + PMB3 + PMB4
  HV   =~ HV1 + HV2 + HV3 + HV4
  ELOY =~ ELOY1 + ELOY2 + ELOY3

  # Structural model (dengan label untuk mediasi)
  HV   ~ a*PMB
  ELOY ~ c*PMB + b*HV

  # Defined parameters
  indirect := a*b
  total    := c + a*b
'

Estimasi Full SEM dengan Bootstrap

sem_fit <- sem(sem_model, data = dat, se = "bootstrap", bootstrap = 1000)

Note

Bootstrap SE digunakan agar confidence interval untuk efek mediasi (indirect effect) tidak bergantung pada asumsi normalitas distribusi sampling.

Common Method Bias (CMB)

Warning

Perhatian: Data dalam penelitian ini dikumpulkan melalui self-report questionnaire dengan sumber yang sama (single-source), sehingga berpotensi terkena Common Method Bias (CMB).

Langkah mitigasi yang dapat dilakukan:

  • Harman’s single-factor test: Jalankan EFA dengan satu faktor; jika varians yang dijelaskan < 50%, CMB tidak menjadi masalah serius
  • Common Latent Factor (CLF): Tambahkan satu faktor laten umum ke model CFA
  • Procedural remedies: Gunakan sumber data berbeda, jaminan anonimitas, counterbalancing item order

CMB bukan berarti hasil tidak valid, tetapi harus dilaporkan sebagai limitasi dan diuji.

Pelaporan CB-SEM (Template APA)

Model pengukuran (CFA) menunjukkan fit yang memadai (\(\chi^2\)(df) = …, p = …; CFI = …; TLI = …; RMSEA = …; SRMR = …). Semua standardized factor loadings berada di atas 0.50, dengan AVE > 0.50 dan CR > 0.70 untuk seluruh konstruk, menunjukkan convergent validity dan reliabilitas yang baik. Discriminant validity terpenuhi berdasarkan kriteria Fornell-Larcker.

Model struktural menunjukkan bahwa PMB berpengaruh positif signifikan terhadap HV (\(\beta\) = …, p < …) dan terhadap ELOY (\(\beta\) = …, p < …). HV juga berpengaruh positif signifikan terhadap ELOY (\(\beta\) = …, p < …). Analisis mediasi dengan bootstrap (1000 resamples) menunjukkan efek tidak langsung PMB → HV → ELOY yang signifikan (indirect = …, 95% CI […, …]), mengindikasikan mediasi parsial.

Bagian B: PLS-SEM dengan seminr

Mengapa PLS-SEM?

  • Tidak memerlukan asumsi distribusi normal
  • Bisa menangani sampel kecil
  • Mendukung indikator formatif dan reflektif
  • Cocok untuk model yang kompleks dengan banyak variabel
  • Fokus pada prediksi, bukan hanya konfirmasi teori
  • Sangat populer di bidang manajemen, pemasaran, dan sistem informasi

Reflektif vs Formatif

Reflektif (Mode A)

  • Indikator adalah efek dari konstruk
  • Menghapus satu indikator tidak mengubah makna konstruk
  • Indikator saling berkorelasi tinggi
  • Contoh: item kepuasan pelanggan

Formatif (Mode B)

  • Indikator membentuk konstruk
  • Menghapus satu indikator mengubah makna konstruk
  • Indikator tidak harus berkorelasi
  • Contoh: SES (pendidikan + pendapatan + pekerjaan)

Note

Model ECSI yang kita gunakan hari ini menggunakan semua konstruk reflektif (Mode A).

Catatan tentang Indikator

Warning

CUSCO hanya memiliki 1 indikator (single item). Ini berarti:

  • Tidak dapat menghitung reliabilitas internal (Alpha, CR)
  • AVE otomatis = 1.0 (hanya satu loading)
  • Interpretasi harus hati-hati — error variance tidak dapat diestimasi secara terpisah

Note

PERV hanya memiliki 2 indikator. Konstruk dengan 2 indikator masih dapat diidentifikasi dalam PLS-SEM, tetapi memiliki keterbatasan dalam estimasi reliabilitas. Gunakan rhoA sebagai ukuran reliabilitas yang lebih robust.

Ringkasan

Apa yang Sudah Kita Pelajari

  • Konsep SEM: variabel laten, measurement vs structural model
  • CB-SEM dengan lavaan:
    • Two-step approach (CFA → Full SEM)
    • Fit indices, CR, AVE, Fornell-Larcker
    • Analisis mediasi dengan bootstrap
  • PLS-SEM dengan seminr:
    • Outer model: loadings, reliabilitas, HTMT, Fornell-Larcker, cross-loadings
    • Inner model: path coefficients, bootstrap, \(R^2\), \(f^2\)
    • Mediasi dan PLS-Predict
