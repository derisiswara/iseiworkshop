Dataset

Workshop ini menggunakan 3 dataset untuk 3 sesi yang berbeda.

1. Regresi: World Values Survey (WVS) Wave 7

Untuk sesi Pengantar R dan Analisis Regresi, kita menggunakan data dari World Values Survey Wave 7 yang telah diterjemahkan ke Bahasa Indonesia. Data tersedia dalam format Excel (.xlsx).

File Data

File Negara n
wvs1.xlsx Singapura, Kanada, Selandia Baru 7.087
wvs2.xlsx Singapura, Thailand, Indonesia 6.712

Kamus Data

Variabel Identitas

Variabel Tipe Deskripsi
negara Karakter Kode negara
id_responden Numerik Identifikasi unik setiap responden

Variabel Pentingnya Aspek Kehidupan (Skala 1–4, 1 = sangat penting)

Variabel Deskripsi
pentingnya_keluarga Seberapa penting keluarga dalam hidup responden
pentingnya_teman Seberapa penting teman dalam hidup responden
pentingnya_waktu_luang Seberapa penting waktu luang
pentingnya_pekerjaan Seberapa penting pekerjaan

Variabel Persepsi dan Kepuasan (Skala 1–10)

Variabel Deskripsi
kebebasan_memilih Persepsi kebebasan memilih dan mengontrol hidup (10 = paling bebas)
kepuasan_hidup Kepuasan hidup secara keseluruhan (10 = paling puas)
kepuasan_finansial Kepuasan terhadap kondisi keuangan (10 = paling puas)

Variabel Politik dan Agama

Variabel Deskripsi
skala_politik Orientasi politik kiri–kanan (1 = kiri, 10 = kanan)
religiusitas Tingkat religiusitas (1–10)

Variabel Demografi

Variabel Tipe Deskripsi
jenis_kelamin Karakter Laki-laki atau Perempuan
tahun_lahir Numerik Tahun lahir responden
usia Numerik Usia responden
status_pernikahan Karakter Status pernikahan
status_pekerjaan Karakter Status pekerjaan

Referensi

Haerpfer, C., Inglehart, R., Moreno, A., Welzel, C., Kizilova, K., Diez-Medrano, J., Lagos, M., Norris, P., Ponarin, E. & Puranen B. (2022): World Values Survey Wave 7 (2017-2022) Cross-National Data-Set. Version: 4.0.0. World Values Survey Association. DOI: doi.org/10.14281/18241.18
