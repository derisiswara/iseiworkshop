Dataset untuk Sesi 1 & 2: World Values Survey (WVS)

Untuk sesi Pengantar R dan Analisis Regresi, kita menggunakan data dari World Values Survey Wave 7 yang telah diterjemahkan ke Bahasa Indonesia.

File Data

File Negara Keterangan
wvs1.csv Singapura, Kanada, Selandia Baru Digunakan untuk latihan sesi 1 & 2

Kedua file sudah tersedia di folder data/ dalam repositori workshop. Jika Anda mengunduh secara terpisah, simpan di dalam folder data/ pada project RStudio Anda.

CatatanApa itu CSV?

CSV (Comma-Separated Values) adalah jenis file yang menyimpan data dalam format teks biasa. Setiap baris mewakili satu baris data, dan setiap kolom dipisahkan oleh koma. Format ini umumnya digunakan untuk menyimpan dan mentransfer data. Anda dapat membuka file CSV di Excel, Google Sheets, atau bahkan Notepad.

Dataset untuk Sesi 3: SEM

Sesi SEM menggunakan 2 dataset dummy yang dirancang dengan dasar teori yang kuat.

CB-SEM: sem_ecommerce.csv

Dataset kepuasan pelanggan e-commerce (n = 300) berdasarkan teori ACSI, SERVQUAL, dan Trust.

Konstruk Kode Indikator Teori
Kualitas Layanan SQ SQ1–SQ4 SERVQUAL (Parasuraman et al.)
Persepsi Nilai PV PV1–PV3 Perceived Value
Kepercayaan TR TR1–TR4 Trust Theory (McKnight et al.)
Kepuasan Pelanggan CS CS1–CS4 ACSI (Fornell et al.)
Loyalitas Pelanggan CL CL1–CL3 Customer Loyalty
Niat Beli Ulang RI RI1–RI3 Repurchase Intention

Semua konstruk reflektif (Mode A). Skala Likert 1–7.

PLS-SEM: sem_innovation.csv

Dataset inovasi dan kinerja UMKM (n = 200) berdasarkan Resource-Based View dan Market Orientation.

Konstruk Kode Mode Indikator Teori
Kapabilitas Inovasi IC Formatif (B) IC1–IC4 RBV (Barney, 1991)
Orientasi Pasar MO Formatif (B) MO1–MO3 Narver & Slater (1990)
Orientasi Kewirausahaan EO Formatif (B) EO1–EO3 Lumpkin & Dess (1996)
Keunggulan Bersaing CA Reflektif (A) CA1–CA4 Competitive Advantage
Kinerja Perusahaan FP Reflektif (A) FP1–FP4 Firm Performance

Skala Likert 1–7. Juga mengandung variabel kontrol: firm_age, firm_size, industry.
